Unlimited collaborators in Github private repositories

• প্রত্যেক প্রোগ্রামার ও ইঞ্জিনিয়ারদের জন্য github স্বর্ণের খনির মতো।এখানে খুব সহজেই একজন নিজের প্রজেক্ট হোস্ট করে রাখতে পারেন আবার অন্যের হোস্ট করা রিসোর্স নিজের প্রজেক্টে ব্যবহার করতে পারেন।Github হলো বিশ্বের সবচেয়ে বড় ওপেন সোর্সের কারখানা।এখানে আপনি আপনার প্রজেক্ট Public or Private ভাবে হোস্ট করতে পারেন ।তবে সমস্যা ছিল private repositories -, Github Pro user ছাড়া private repositories তৈরী করা যেত না এবং প্রাইভেট রিপোজিটরিতে সামান্য সংখ্যক সহযোগী এড করা যেত।তবে কিছু দিন আগে গিটহাব তাদের private repositories সাধারণ ব্যবহারকারীদের জন্য ফ্রী করে দিয়েছে লাইফ টাউমের জন্য এবং কিছু Paid tools এর মুল্য হ্রাস করে দিয়েছে। সেগুলো নিচে হুবহু তুলে ধরা হলো।

We introduced a few changes to make all of the core GitHub features free for everyone. Private repositories with unlimited collaborators are now available for all GitHub accounts, and we’re reducing prices for some of our paid plans. The details
1•We introduced a new GitHub Free plan for teams with unlimited collaborators in private repositories, 2,000 GitHub Actions minutes/month, and GitHub Community Support.
2•You now have unlimited collaborators in private repositories as part of your GitHub Free plan. Monthly pricing for our Pro plan has been reduced from $7/month to $4/month.
3•Monthly pricing for our paid Team plan has been reduced from $9/user to $4/user.
4•Actions minutes and Packages storage will continue to be free for all public repositories.
